# swa

## Project setup
```
npm install
```

move /dbs_template/config.js to /config.js  
rename /dbs_template to /dbs

### Compiles and hot-reloads for development
```
npm run serve
node koa_swa.js
```

### Compiles and minifies for production
```
npm run build
```

### Run production
```
node koa_swa.js
```
visit [http://localhost:8086](http://localhost:8086)
